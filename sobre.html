Sobre

Somos uma associação de desenvolvimento de projetos de programação e design. Temos sistemas prontos e projetos em desenvolvimento.

Na Pysi, transformamos idéia em realidade, problemas em soluções, organização em agilidade. Nosso foco são projetos personalizados para pessoas físicas, pequenas e grandes empresas .

Estamos disponibilizando links para downloads de algumas de nossas ferramentas, como controle de contas a pagar, controle de cheques, coisas simples para ajudar a organizar sem gastar nada.

Fazemos projetos personalizados de desenvolvimento também, utilizando as tecnologias mais recentes para isso. Nossos projetos são desenvolvidos utilizando por exemplo:

  • Microsoft .Net
  • Visual Basic .Net
  • Visual C# .Net
  • ASP.Net
  • ASP
  • Ajax
  • Programação visual com estilos (CSS)
  • Tableless
  • Web 2.0
  • Dentre outras tecnologias

Se você deseja fazer um projeto personalizado, entre em contato conosco para um orçamento
